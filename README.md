# Shivay Medical Inventory

Offline Android inventory app.

## Features
- Manual medicine entry
- A-Z search by brand, composition, or company
- Manual stock +/- controls
- Low-stock filter (5 units or fewer)
- Expiry filter (within 3 months or expired)
- Batch, MRP, company, composition and rack fields
- Offline local storage

## Build APK without Android Studio
Upload all project files to a GitHub repository. Open Actions > Build Android APK > Run workflow.
After the build finishes, open the run and download the Shivay-Medical-Inventory-APK artifact.
